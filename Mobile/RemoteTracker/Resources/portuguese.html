RemoteTracker



O que é esse software?

Este software é um modo simples de você rastrear seu aparelho se você o perder, ser roubado ou usar o aparelho como um dispositivo de segurança para seu carro por examplo. Existem várias utilidades para o RemoteTracker, use sua imaginação! Mas lembre-se: você não pode usar o RemoteTracker para rastrear pessoas sem sua aprovação.

Como o RemoteTracker funciona?

RemoteTracker funciona interceptando mensagens SMS formatadas (Mensagens de Texto) enviadas a partir de qualquer celular. Se você deseja saber onde está o seu aparelho, você deve usar qualquer outro aparelho e enviar um SMS para seu aparelho desaparecido.

O SMS que você enviará deverá iniciar com RT# seguido por alguns parâmetros listados adiante.

O SMS será interceptado pelo RemoteTracker conterá comandos que permitirão que você controle seu telefone remotamente. Existem comandos básicos e avançados. Você irá usá-los de acordo com seu conhecimento.

Após enviar o comando, o RemoteTracker irá processar o comando e enviará uma resposta para você. Esta resposta pode ser outro SMS, um E-Mail ou um arquivo envido para sua conta de FTP, se você possuir alguma.

É importante dizer: RemoteTracker funciona de modo silencioso. Não há interface informando que existe algum comando sendo processado.

Função de Resgate (Anti Sequestro)

RemoteTracker pode ajudá-lo a preservar sua própria segurança. Você pode configuar até quatro número de celulares e quatro endereços de e-mail para serem usados quando você tiver problemas e precisar de ajuda. Aqui você deverá informar os números de sua esposa, mãe, pai, ou qualquer outro que você possa contar num momento de necessidade.

Em Programas você encontrará um aplicativo chamado TopSecret. Após iniciado este programa irá solicitar que você informe uma senha. Isso é falso e você não precisa digitar nada. Após 30 segundos, se você não clicar em Cancel, uma mensagem pré definida será enviada para os quatro celulares e para os quatro e-mails cadastrados como emergência. A mensagem enviada conterá o seu nome e a posição informada pelo GPS do seu aparelho.

Controle Remoto

Também em Programas você encontrará o aplicativo chamado RTRemote. Este aplicativo tem dois objetivos:

1- Testar o RemoteTracker em seu próprio aparelho sem pagar por SMS;

2- Enviar comandos para outros aparelhos com o RemoteTracker instalado. Dessa forma evita-se a digitação de mensagens SMS, pois o RTRemote faz isso para você.

Proteção do Cartão SIM

Um grande problema quando você instala um software como o RemoteTracker é: se o cartão SIM for trocado, você não consegue mais rastrear o aparelho porque você não sabe qual é o novo número!

Não! Sempre que seu aparelho iniciar o cartão SIM será verificado se ele está na lista de cartões conhecidos. Você pode configurar até quatro cartões SIM. Se alguém instalar algum cartão desconhecido, então você receberá uma mensagem alertando dessa troca. A mensagem será acompanhada do novo número e você poderá continuar rastreando seu aparelho.

O RemoteTracker é Seguro?

O RemoteTracker possui algumas funcionalidades para preservar a segurança do aplicativo:

  • Proteção do Cartão SIM;
  • Modo escondido. O RemoteTracker não é exibido na lista de aplicativos que podem ser desinstalados e é movido para outra pasta;
  • Senha. Você somente poderá desinstalar o RemoteTracker ou processar comandos se você informar a senha.

Configurando o RemoteTracker

Se você está lendo esta Ajuda, provavelmente você iniciou o aplicativo Config na pasta Programas. Se não, por favor inicie o aplicativo.
Clique em Menu para ver a lista de itens que você poderá configurar:
  • Opções: aqui você tem as opções para controlar o comportamento do RemoteTracker:
    • Idiomas: você pode escolher o melhor idioma para você. Existem alguns idiomas disponíveis e você pode fazer a sua própria tradução. Por favor entre em contato com o autor se desejar.
    • Modo de Depuração: esta opção será usada se você precisar reportar alguma falha no aplicativo. Quando marcada, cada comando processado será avaliado e um arquivo contendo informações úteis para que o autor possa efetuar a correção do problema. Envie esse arquivo quando você entrar em contato com o autor.
    • Deslitar a Tela ao Processar: marcada por padrão, esta opção fará com que o aparelho tenha sua tela desligada quando um comando é processado pelo RemoteTracker, deixando o aplicativo em modo escondido.
    • Ativar o TopSecret: desabilitado por padrão, esta opção permitirá que você use o aplicativo TopSecret.
    • Senha: você pode definir uma senha. Esta senha será uma importação opção porque será utilizada para processar comandos, acessar o aplicativo Config e para desinstalar o RemoteTracker. Sem uma senha qualquer pessoa poderá enviar comandos para seu aparelho.
    • Configuração do E-Mail: como descrito anteriormente, RemoteTracker pode responder aos comandos por e-mail. Mas você deve configurar uma conta de e-mail para ser utilizada para enviar as mensagens.
    • Número de tentativas do GPS: qualquer comando que utiliza o GPS irá fazer com que o mesmo tente se conectar com os satélites. Mas, se o aparelho estiver dentro de construções ele não conseguirá essa conexão. Para evitar que a bateria seja drenada, RemoteTracker tem um limite de tentativas.
    • Tempo para os Comandos ELT/FLT: Os comandos especiais ELT/FLT usam o GPS para gravar o percurso percorrido pelo portador do aparelho por um período de tempo. Após esse tempo um arquivo KMZ é enviado para seu e-mail ou servidor de FTP. O arquivo deve ser aberto usando o Google Earth.
    • Definir o Som do Alarme: você pode escolher um som diferente para ser tocado quando o RemoteTracker processar o comando ALARM.
    • Configurações de FTP: se você possuir um conta pessoal em servidor FTP, você poderá configurar essa conta aqui para que o RemoteTracker possa responder a comandos que utilizam esse tipo de resposta. A resposta será fornecida como um arquivo e transferido para sua conta. Se você não possui uma conta desse tipo, deixe essa configuração em branco.
    • Configurações do GPS: existem duas formas de acessar o GPS: usar o GPS administrado pelo Windows Mobile oo usar uma configuração manual de porta COM (serial) e taxa de transferência.
    • Esconder o RemoteTracker: esta opção uma vez disparada não poderá ser desfeita. RemoteTracker irá se mover para outra pasta e não aparecerá mais na lista de aplicativos que podem ser desinstalados. Tenha certeza de que fez todas as configurações corretamente antes de usar essa função.
  • Cartões SIM: você poderá definir até quatro cartões SIM conhecidos. Se você, ou alguém, trocar o cartão por algum cartão desconhecido, seus números e e-mails de emergência receberão uma mensagem alertando da troca e receberão o novo número do celular.
  • Celulares para Emergência: aqui você poderá configurar até quatro números de celulares que serão usados em casos de emergência. Seja pela troca de um cartão SIM desconhecido ou seja pelo uso do aplicativo TopSecret.
  • E-Mails para Emergência: aqui você poderá configurar até quatro endereços de e-mail que serão usados em casos de emergência. Seja pela troca de um cartão SIM desconhecido ou seja pelo uso do aplicativo TopSecret.
  • Mensagem de Emergência: você pode criar sua própria mensagem de ajuda para ser enviada quando você utilizar o aplicativo TopSecret.
  • Registro de Comandos: cada comando recebido e processado é salvo em um arquivo criptografado. Você poderá ver essa lista de comandos processados utilizando essa opção.
  • Aplicar Configurações: esta opção salva a configuração sem sair da aplicação.
  • Ajuda: exibe essa tela de ajuda.
  • Sobre: exibe informações sobre o aplicativo.
Como Enviar Comandos ao RemoteTracker?

Para enviar comandos ao RemoteTracker você tem que enviar um SMS para seu aparelho. Este SMS deve seguir a seguinte sintaxe:

rt#[comando]#[número do celular ou endereço de e-mail]#[senha]

Onde:
  • rt# é o cabeçalho. O Windows Mobile irá entender que deve inicar o RemoteTracker porque a mensagem começa com esse cabeçalho;
  • [comando] é o comando a ser processado. Por favor veja a lista abaixo;
  • [número do celular ou endereço de e-mail] isso é opcional. Se for um comando que responda por SMS, você pode definir um número de celular que irá receber a resposta. Se deixado em branco o celular que enviou a mensagem irá receber a resposta. Se o comando responder por e-mail você poderá definir um endereço para receber a resposta. Se você deixar em branco, o endereço de resposta padrão será utilizado;
  • [senha] este parâmetro é opcional, mas se você definir uma senha, então o parâmetro passa a ser obrigatório;
  • # este é o caracter separador. Você deve informar esse separador entre os parâmetros.
Exemplos:
  • rt#[comando] - irá processar o [comando] e responder por SMS/E-Mail/FTP a quem enviou o comando, sem senhas;
  • rt#[comando]#12345678 - irá processar o [comando] e responder por SMS para 12345678, sem senhas;
  • rt#[comando]#12345678#1234 - irá processar o [comando] e responder por SMS para 12345678 usando 1234 como senha;
  • rt#[comando]##1234 - irá processar o [comando] e responder por SMS/E-Mail/FTP a quem enviou o comando usando 1234 como senha. Note o duplo #.
Notas:
  • Todos os comandos que iniciam com a letra E são comandos que respondem por e-mail;
  • Todos os comandos que iniciam com a letra F são comandos que respondem por FTP;
  • Existem alguns comandos que necessitam de parâmetros extras. Neste caso a sintaxe é: rt#[comando,parametros]#[número ou e-mail]#[senha]
Comandos Básicos
    help - envia a lista de comandos por SMS
    cb - faz uma chamada para o número informado/origem do comando e deixa o microfone aberto
    gp - inicia o GPS e responde com as coordenadas
    pb - envia lista com todos os contatos armazenados no cartão SIM
    go - pega as informações do proprietário. Estas informações são definidas em Iniciar/Configurações
    rst - reinicia seu dispositivo
    findme - este comando é útil para você informar a amigos onde você está. Seu amigo deverá enviar um SMS com este comando para seu aparelho, então o RemoteTracker irá perguntar se você deseja enviar sua posição. Respondendo positivamente o GPS será ativado e seu amigo receberá um SMS com sua posição atual. Este comando não utiliza senhas
    alarm - apenas toca um som. Particularmente útil quando você não se lembra onde deixou o aparelho em sua própria casa ou escritório
    gi - pega o número IMEI do aparelho e o número IMSI do cartão SIM
Comandos Básicos por E-Mail
    elt - grava o caminho percorrido pelo portador do aparelho e envia por e-mail um arquivo KML para ser aberto pelo Google Earth
    ehelp - mesmo que 'help', por e-mail
    egp - mesmo que 'gp', por e-mail
    epb - mesmo que 'pb', por e-mail
    ego - mesmo que 'go', por e-mail
    egi - mesmo que 'gi', por e-mail
Comandos Avançados
    dsc - apaga os contatos do cartão SIM
    dkz - apaga os arquivos KMZ
    ganfl - envia uma lista de todas as ligações recebidas ou originadas se você tiver o programa VITO Audio Notes instalado
    gip - envia o endereço IP atual. Se seu aparelho não possuir um endereço IP, então uma conexão com a internet será realizada
    vnc - inicia o servidor VNC
    listapp - lista todos os aplicativos instalados no aparelho
    runapp - executa um determinado aplicativo. Este comando utiliza uma sintaxe especial: rt#runapp,[Aplicativo]##[senha]
    cellid - busca informações da torre celular que o aparelho está respondendo no momento
    lock - trava o aparelho. Este comando utiliza uma sintaxe especial: rt#lock,[mensagem de travamento]##[senha] . TOME CUIDADO COM ESSE COMANDO! NÃO O UTILIZE SE VOCÊ NÃO PUDER USAR O COMANDO UNLOCK!
    unlock - destrava o aparelho
    msg - exibe uma caixa de diálogo no aparelho com uma mensagem. Este comando utiliza uma sintaxe especial: rt#msg#[texto a exibir]#[senha]
    callhist - retorna a lista de ligações recebidas e originadas. Por padrão esse comando retorna as últimas 5 chamadas, mas você pode definir a quantidade de chamadas que deseja ver. Este comando utiliza uma sintaxe especial: rt#callhist,[número de chamadas]##[senha]
Comandos Avançados por E-Mail
    eganfl - mesmo que 'ganfl', por e-mail
    eganf - envia todas as gravações de ligações recebidas ou originadas se você tiver o programa VITO Audio Notes instalado. Este comando utiliza uma sintaxe especial: rt#eganf,[número de gravações]##[senha]
    egip - mesmo que 'gip', por e-mail
    elistapp - mesmo que 'listapp', por e-mail
    ecellid - mesmo que 'cellid', por e-mail
    ecallhist - mesmo que 'callhist', por e-mail
Comandos que usam FTP
    ftp - envia arquivos para o servidor FTP. Este comando utiliza uma sintaxe especial: rt#ftp,[arq1],[arq2],[arq3]##[senha]
    fhelp - mesmo que 'help', por FTP
    fgp - mesmo que 'gp', por FTP
    fpb - mesmo que 'pb', por FTP
    flt - mesmo que 'elt', por FTP
    fgo - mesmo que 'go', por FTP
    fgi - mesmo que 'gi', por FTP
    fganfl - mesmo que 'ganfl', por FTP
    fganf - mesmo que 'eganf', por FTP
    fgip - mesmo que 'gip', por FTP
    flistapp - mesmo que 'listapp', por FTP
    fcellid - mesmo que 'cellid', por FTP
    fcallhist - mesmo que 'callhist', por FTP
Comandos que se integram ao MortScript
    mslist - envia lista com todos os arquivos do MortScript encontrados
    emslist - mesmo que 'mslist', por e-mail
    fmslist - mesmo que 'mslist', por FTP
    msrun - executa scripts do MortScript. Este comando utiliza uma sintaxe especial: rt#msrun,[script1],[script2],[script3]##[senha]
