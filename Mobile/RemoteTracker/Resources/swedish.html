RemoteTracker



Vad gör programmet?

Programmet utgör ett enkelt sätt att hitta din enhet om du tappar den, får den stulen eller om du av säkerhetsskäl t.ex. vill följa din bil. Hur du kan använda RemoteTracker, begränsas, nästan, bara av din fantasi! Men glöm inte - du får inte använda RemoteTracker för att följa personer utan deras godtycke!

Hur funkar det?

RemoteTracker uppfattar SMS (text meddelanden) skickade till enheten. Om du vill veta var din enhet är, använder du en annan enhet och skickar ett SMS till den saknade enheten.

SMS:et du skickar skall börjar på RT# följt av parameterar enligt nedan.

Genom texten i SMS:et kan du enkelt styra din enhet i fjärr. Det finns enkla och mer avancerade kommandon, använd dem efter behov.

När du skickat ett kommando, kommer RemoteTracker att bearbeta det och skicka dig ett svar, antingen som ett SMS, e-post eller en fil till ditt FTP konto, om du har ett.

Viktigt: RemoteTracker arbetar i tyst mode. Inga fönster visas när ett kommando bearbetas.

Räddnings (anti kidnappning) funktion

RemoteTracker kan hjälp dig med din egen säkerhet. Du kan ställa in upp till fyra mobiltelefonnummer och fyra e-postadresser som skall användas när du har problem och behöver hjälp. Här fyller du i information om din fru, mor, far osv.

I Program hittar du applikationen TopSecret. När du startar detta program frågas du efter ett lösenord, detta är dock fake och du behöver inte skriva något, efter 30 sekunder, om du inte avbryter, skickas det fördefinierade meddelande till dina nödnummer och nöd epost följt av ditt namn och nuvarande position baserad på din GPS information.

Fjärr kontroll

I Program hittar du också applikationen RTRemote. Detta program har två uppgifter:

1- Testa RemoteTracker på din egen enhet utan att betala för SMS;

2- Skicka kommandon till en annan enhet med RemoteTracker utan SMS. RTRemote gör det för dig.

SIM korts bytes skydd

Ett stort problem när du installerar ett program som RemoteTrackerär: om SIM kortet byts, förloras allt!

Nej! Varje gång din enhet startas verifieras det installerade SIM kortet om det är ett känt eller okänt kort. Upp till fyra kort kan definieras. Om någon installeras ett okänt SIM kort, skickas ett meddelande om bytet samt det nya mobilnummret till dig för att kunna följa enheten.

Är RemoteTracker säker?

RemoteTracker har några funktioner för att bevara din säkerhet:

  • SIM korts bytes skydd;
  • Gömt läge. RemoteTracker visas inte i avinstallationslistan och flyttas till en annan mapp;
  • Lösenord. Du kan bara avinstallera RemoteTracker eller svara på ett kommando när angivit/skickat lösenordet.

Konfiguera RemoteTracker

Om du läser denna hjälp, har du förmodligen startat programmet Config i Program mappen. Om inte, starta denna applikation.
Klicka på Meny för att se vad som kan konfigueras:
  • Inställningar: här har du inställningarna för att styra RemoteTrackers uppförande:
    • Språk: du kan välja språk efter eget tycke. Flera språk finns tillgängliga, och du kan även göra en egen översättning. Skicka ett e-post till utvecklaren om detta.
    • Debug läge: använd detta val om du vill rapportera ett fel. När detta är valt, loggas varje kommando och skrivs i en fil. Skicka denna fil till utvecklaren.
    • Släck skärm under körning: defaultvald, detta val låter din enhet bearbeta ett kommando i gömt läge.
    • Aktivera TopSecret: ej defaultvald, detta val låter dig använda TopSecret applikationen.
    • Lösenord: du kan ange ett lösenord. Detta är ett väldigt användbart val, då detta lösenord används för att bearbeta kommandon, för att starta Config och att avinstallera RemoteTracker. Utan lösenord kan alla styra din enhet.
    • E-post konfiguration: som beskrivet ovan, RemoteTracker kan svara på kommandon med e-post. Men du måste ställa in e-post kontot för att skicka e-post.
    • GPS försök för GPS kommando: alla GPS kommandon kommer att försöka att koppla upp sig mot sateliter via GPS radion. Men, om enhet är inne i en byggnad kan den inte få kontakt. För att förhindra att batteriet töms, har RemoteTracker ett begränsat antal försök att koppla upp sig.
    • Timeout för ELT/FLT kommando: ELT/FLT är speciella GPS kommandon för att spela in vägen din enhet rör sig och skicka en KMZ fil till dig med e-post/FTP. Denna fil kan öppnas i Google Earth.
    • Definiera alarm ljud: you kan välja olika ljudfiler som RemoteTracker skall spela upp när ALARM kommandot bearbetas.
    • FTP konfiguration: om du har ett personligt FTP konto, kan du ställa in det här och RemoteTracker kan skicka filer till FTP servern. Om du inte vet vad FTP är, lämna den tom.
    • GPS konfiguration: det finns två sätt att används din GPS: Hanterad av Windows eller Manuellt via COM-port och baudrates val.
    • Göm RemoteTracker: detta val kan inte ångras!. RemoteTracker flyttas till en annan mapp och tas bort från Program mappen och avinstallationslistan. Var säker på att du gjort alla konfigurationer innan du väljer detta val.
  • SIM kort: upp till fyra SIM kort kan definieras. Om du, eller någon annan, byter till ett okänt SIM kort kommer dina nöd nummer och nöd epost att få ett meddelande om bytet och det nya nummret.
  • Nödnummer: ange upp till fyra mobiltelefonnummer som meddelas när ett okänt SIM kort sätts i enheten, eller om du startar TopSecret applikationen.
  • Nöd e-post: ange upp till fyra e-postadresser som meddelas när ett okänt SIM kort sätts i enheten, eller om du startar TopSecret applikationen.
  • Nöd meddelande: skapa ditt eget meddelande som skickas när du startar TopSecret.
  • Se kommandologgen: varje mottaget kommando sparas i en krypterad fil. Detta val visa dig vad RemoteTracker svarade.
  • Verkställ konfiguration: spara konfigurationen utan att avsluta programmet.
  • Hjälp: visar denna hjälp.
  • Om: visar ett om fönster.
Skicka kommando till RemoteTracker

För att skicka ett kommando till RemoteTracker, behöver du skicka ett SMS till din enhet. Detta SMS måste ha exakt denna syntax:

rt#[kommando]#[mobilnummer eller e-post adress]#[lösenord]

Där:
  • rt# är huvudet. Windows Mobile vet då att RemoteTracker skall startas;
  • [kommando] är kommandot som skall beartberas. Se listan nedan;
  • [mobilnummer eller e-post adress] valfritt. Om kommandot skall svaras med SMS, kan du ange mottagande mobilnummret. Om blank, skickas meddelandet till uppringaren. Om kommandot skall svaras med e-post, kan du ange mottagande e-post adress. Om blank, används default e-post adress;
  • [lösenord] valfritt, men om definierad måste det anges.
  • # detta tecken är separator. Du måste ange det mellan parametrar.
Exempel:
  • rt#[kommando] - bearbetar [kommando] och svara med SMS till uppringaren, inget lösenord;
  • rt#[kommando]#12345678 - bearbetar [kommando] och svara med SMS till 12345678, inget lösenord;
  • rt#[kommando]#12345678#1234 - bearbetar [kommando] och svara med SMS till 12345678 med 1234 som lösenord;
  • rt#[kommando]##1234 - bearbetar [kommando] och svara med SMS till uppringaren med 1234 som lösenord. Observera de dubbla #.
Noteringar:
  • Alla kommandon som startar med E svaras med e-post;
  • Alla kommandon som startar med F svaras till FTP;
  • Vissa kommandon kräver extra parametrar. I detta fall är syntaxen: rt#[kommando,parameter]#[mobilnummer eller e-post adress]#[lösenord]
Enkla kommandon
    help - skicka listan över kommandon med SMS
    cb - ring ett samtal till ett nummer och låt mikrofonen vara på
    gp - starta GPS:n och skicka koordinater
    pb - skicka alla kontakter på SIM kort
    go - skicka ägar information. Dessa anges i Start/Settings.
    rst - mjukstarta enheten
    findme - detta kommando är användart för att tala om för vänner var du är. Din vän skickar detta kommando till dig och din enhet frågar dig om du vill svara med koordinater. Detta kommando använder inte lösenord.
    alarm - spela en ljudfil. Användbart om du förlagt din enhet t.ex. hemma...
    gi - skicka enhetens IMEI nummer och det installerade SIM kortets IMSI nummer
E-post enkla kommandon
    elt - logga rutten din enhet färdas och skicka en KML fil med e-post för att visas i Google Earth
    ehelp - samma som 'help', med e-post
    egp - samma som 'gp', med e-post
    epb - samma som 'pb', med e-post
    ego - samma som 'go', med e-post
    egi - samma som 'gi', med e-post
Avancerade kommandon
    dsc - ta bort SIM kontakter
    dkz - ta bort KMZ filer
    ganfl - skicka en lista över alla inkommande och utgående samtal om du har VITO Audio Notes installerad.
    gip - skicka nuvarande IP adress. Om din enhet inte har en IP, kopplas en internet anslutning upp.
    vnc - starta VNC server
    listapp - lista alla installerade applikationer
    runapp - kör en definierad applikation. Detta kommando använder special syntaxen: rt#runapp,[applikation att köra]##[lösenord]
    cellid - skicka information från nuvarande cell torn
    lock - lås din enhet. Detta kommando använder special syntaxen: rt#lock,[personal lock message]##[lösenord] . VAR FÖRSIKTIG MED DENNA KOMMANDO. ANVÄND DET INTE OM DU INTE KAN ANVÄNDA UPPLÅSNINGSKOMMANDOT!
    unlock - lås upp din enhet
    msg - visa ett meddelandefönster för användaren. Detta kommando använder special syntaxen: rt#msg#[text att visa]#[lösenord]
    callhist - skicka samtalshistoriken. Detta kommando använder special syntaxen: rt#callhist,[antalet samtal]##[lösenord]
E-post avancerade kommandon
    eganfl - samma som 'ganfl', med e-post
    eganf - skicka en lista över alla inkommande och utgående samtal om du har VITO Audio Notes installerad. Detta kommando använder special syntaxen: rt#eganf,[Antal poster]##[lösenord]
    egip - samma som 'gip', med e-post
    elistapp - samma som 'listapp', med e-post
    ecellid - samma som 'cellid', med e-post
    ecallhist - samma som 'callhist', med e-post
FTP kommandon
    ftp - skicka filer till en konfiguerad server. Detta kommando använder special syntaxen: rt#ftp,[fil1],[fil2],[fil3]##[lösenord]
    fhelp - samma som 'help', till FTP konto
    fgp - samma som 'gp', till FTP konto
    fpb - samma som 'pb', till FTP konto
    flt - samma som 'elt', till FTP konto
    fgo - samma som 'go', till FTP konto
    fgi - samma som 'gi', till FTP konto
    fganfl - samma som 'ganfl', till FTP konto
    fganf - samma som 'eganf', till FTP konto
    fgip - samma som 'gip', till FTP konto
    flistapp - samma som 'listapp', till FTP konto
    fcellid - samma som 'cellid', till FTP konto
    fcallhist - samma som 'callhist', till FTP konto
MortScript kommandon
    mslist - lista alla MortScript filer på din enhet
    emslist - samma som 'mslist', med e-post
    fmslist - samma som 'mslist', till FTP konto
    msrun - kör MortScript filer på din enhet. Detta kommando använder special syntaxen: rt#msrun,[script1],[script2],[script3]##[lösenord]
