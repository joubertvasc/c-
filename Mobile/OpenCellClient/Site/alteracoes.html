  • Versão 0.3.1-0
    - Suporte para dispositivos sem touchscreen;

  • Versão 0.3.0-0
    - Nova módulo de controle do GPS;
    - Nova opção para ver uma bússola baseada em GPS;
    - Suporte para novo banco de dados de torres: www.celldb.org;
    - Nova estrutura do banco de dados interno. Agora a força do sinal das torres de celular ou dos access points também são armazenadas;
    - Capacidade para armazenar várias informações para a mesma torre/access point, dependendo do sinal;
    - Adicionada opção para ver a localização de determinada torre/access point em mapa do Google Maps. Necessita conexão web;
    - Removida opção para ver as coordenadas enviadas para OpenCellID, porque não tinha utilidade.

  • Versão 0.2.0-0
    - Nova módulo de controle do GPS;
    - Nova opção para escolher entre o GPS controlado pelo Windows ou configurado manualmente;
    - Novo módulo de Backup/Restore do banco de dados;
    - Várias pequenas correções;
    - Agora o programa também rastreia pontos de rede sem fio (WiFi), por enquanto para montar um banco de dados local;
    - Agora quando uma coordenada é enconrada essa linha é exibida automaticamente;
    - Primeira versão do Termo de Serviço.

  • Version 0.1.1-2
    - Correções.

  • Version 0.1.1-1
    - Correções.

  • Versão 0.1.0-0
    - Adicionado opção para fazer uma cópia de segurança do banco de dados;
    - Adicionado opção para restaurar cópias de segurança;
    - Adicionado opção para reiniciar o dispositivo antes de iniciar a procura por células. Útil em alguns aparelhos;
    - Adicionado opção para baixar da internet a lista completa de coordenadas que foi enviada, desde que você esteja usando sua própria chave do openCellId.org.

  • Versão 0.0.3-0
    - Adicionada um pequeno medidor de bateria na aba de dados;
    - Alterado a forma de gravação dos dados de arquivo XML para o gerenciador de banco de dados SQLServerCE;
    - As opções do menu foram redefinidas;
    - Adicionada opção para ver apenas as novas células;
    - Adicionada opção para ver todas as células;
    - Adicionada opção para apagar todo o histórico de células;
    - Adicionado opção no menu de contexto do grid de dados para excluir uma linha da tabela.

  • Versão 0.0.2-0
    - Correção: wifi estava sendo desligado quando se saía da aplicação;
    - Adicionada opção para reiniciar o rádio GSM antes de iniciar a busca por torres;
    - Adicionada opção para enviar os dados automaticamente para www.opencellid.org;
    - Adicionada opção para evitar o modo hibernação;
    - As larguras das colunas da lista de torres são recuperadas quando se inicia o aplicativo.
